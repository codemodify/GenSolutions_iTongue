4 모드는 쉽게


Travel Mode – 그것은 당신이 여행 여부를 매일 사용, 이동 및 직사각형하지만 당신은 손가락으로 원하는 크기를 조절. 당신은 피사체에 가까이하지 않은 경우에 줌.

Book Mode – 도서 또는 레스토랑 메뉴를 읽기에 유용합니다. 번역 단어, 라인 또는 전체 단락.

Text Mode – 시대에 당신이 이미지를이 없을 때. 그냥 입력 텍스트와 언어를 선택 - 우리가 가진 건 자네가 알아서.

Scanner Mode – 문서를 스캔하고 더 사용하기 위해 귀하의 이메일로 보내주시기 바랍니다.
