Informações Gerais  (NOTE: Automated Translation)


Traduza de qualquer língua (suportado) para qualquer idioma (apoiado) por apenas apontando a câmera para o texto, em seguida, mover, redimensionar e ampliar o rectângulo mágico para pegar a área que você está interessado em usar modos especiais para viajar, ler livros, jornais ou menus de restaurante. Ou use o modo de texto para a tradução manual.


4 modos para tornar mais fácil


Travel Mode – usá-lo todos os dias se você estiver viajando ou não, mover e redimensionar o retângulo como quiser com o seu dedo. Zoom se você não está perto o assunto.

Book Mode – útil para ler livros ou menus de restaurante. Traduza uma palavra, uma linha ou um parágrafo inteiro.

Text Mode – por vezes, quando você não tem uma imagem. Basta digitar o texto e selecione o idioma - nós temos que cobertas.

Scanner Mode – digitalizar um documento e enviá-lo para seu e-mail para uso posterior.

FAQ

*       Ele não reconhece o texto corretamente.

  Às vezes iTongue não será capaz de reconhecer corretamente o texto, para tentar obter uma imagem clara do texto, sem quaisquer outros elementos, certifique-se que há luz suficiente e você está numa posição perpendicular.

*       No entanto, eu tento não reconhecer o texto.

         Você pode editar o texto reconhecido ou usar o modo texto para a tradução manual.

*       Que tipos de texto faz trabalhar?

          iTongue reconhece a maioria das fontes e tipos, porém, não é capaz de reconhecer um texto manuscrito ou texto esticado.
